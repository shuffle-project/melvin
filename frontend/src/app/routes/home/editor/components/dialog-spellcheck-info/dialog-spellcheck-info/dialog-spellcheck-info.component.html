Spellcheck browser info

You must install a dictionary in order to use the Firefox spell checker. Follow the link below for an overview of the available dictionaries.

Once you have added the dictionary, you may need to right-click on the transcript here in the Melvin Editor and activate “Check spelling” in the resulting menu and select the appropriate language under “Languages”.
