Creating Videos

With Melvin, you can record new videos directly in your browser or upload existing ones.

Record Videos in the Browser

  1. Click on “New Project” above your project list and select “Recorder” from the drop-down menu.

  2. You are now on the recorder page. Add your desired input sources here (e.g., microphone, camera, or screen). At least one audio source and one video or screen source are required to start recording.

    The browser will request permission the first time. Allow access when prompted. If no prompt appears, search online for: “[Your browser, e.g., Firefox] [microphone / camera / screen] allow permissions”

  3. The audio level indicator shows whether your microphone is picking up sound.

  4. Start recording by clicking “Start recording”. Please note that you cannot add or remove input sources during the recording.

  5. After clicking “Stop recording”, the “Project Creation” dialog opens automatically. Specify the primary language of the recording to enable file upload and initiate project creation. To save individual files locally, use the download button located beside each file.

    If you experience upload issues, save the video locally and use the “Upload Existing Video” function later (see below).

  6. Once the project is created, your recording will appear in the project list. Processing may take some time depending on the video length.

Upload Existing Videos

  1. Click on “New Project” above your project list and select “Upload File” from the drop-down menu.

  2. In the “New Project” dialog that appears, you can drag and drop files or use the “Select Files” button to upload one or more files from your computer.

  3. For each file, select a category and language.

  4. Check “Use Audio” for the file that should be transcribed and used as the audio track.

  5. Set a project title and click “Create Project”.

  6. Your project will now appear in the project list. Processing may take some time.

Edit a Project

  1. To edit a project transcript, select a project from the project list. Click the project title or the “Edit” button.

  2. You are now in the editor. Click the project name at the top left to rename it.

  3. Edit the transcript directly in the text field. You can navigate the video using the audio waveform.

  4. If a speaker change was not detected, press Enter to create a new paragraph and assign the new speaker. Paragraphs can be merged using Backspace. To change the speaker, click on the name.

Collaborative Editing

  1. Click the “Share” button in the top right corner of the editor.

  2. Select the “Invite collaborators” tab.

  3. Enter the person's email address and click “Invite”. If they already have a Melvin account, they’ll be added immediately. Otherwise, an email invitation will be sent.

  4. Alternatively, use “Copy Link” to generate a shareable link that allows anonymous collaboration on the project.

Add Additional Transcripts

A transcript in the video's primary language is automatically created for each project. To offer subtitles and transcripts in multiple languages, you can add more transcripts to a project.

  1. Click on “Transcription” below the project name in the editor.

  2. Here, you'll see all available transcripts and options to “Add Transcription” or “Edit Transcription”.

  3. Clicking “Edit Transcriptions” opens a dialog showing all current transcripts. You can download, delete, or edit them using the “More” button (three dots).

  4. Use “Add Transcript” to upload or copy transcripts.

Player

  1. Click the “Watch” button in the project list to open the project in the player (in a new browser tab).

  2. Below the video are various buttons to control and configure the player. Hover over a button to see its function, or use the Tab key to navigate to it.

  3. Use the “Language menu” to show/hide and customize the subtitles and the transcript. For more options, click “Settings” to open the settings dialog.

  4. If your project contains multiple videos, you can manage them using the “Video Selection menu” button, located below the video next to the “Languages menu”.

  5. Use the transcript to navigate the video: clicking a sentence jumps to the corresponding point in the video.

Share a Video

You can easily share your videos using the integrated, accessible player:

  1. In the project list, click the “Watch” button for your project. The video will open in a new tab using the player.

  2. Copy the browser URL to share the video.

Alternatively, you can embed the video with the player into your website using an iframe.

  1. In the project list, click the “More” button of the project and select “Share”. The sharing dialog will open.

  2. Under the “Share videoplayer” tab, click “Copy” to copy the embed code to your clipboard.

FAQ

Can I trim or edit recordings?

No, Melvin currently does not offer any editing or trimming functionality. However, recordings can be edited using an external video editor before being uploaded to Melvin. Depending on the development team's capacity and priorities, editing features may be added in future versions of Melvin.

Can I blur the background of my webcam in the recorder?

Currently, this functionality is not supported by Melvin or web browsers.

However, you can use external software to blur your webcam background before recording. For example, you can use OBS Studio with a virtual camera or other webcam software that supports background blurring.

If you're using a Mac with Apple silicon and macOS Sequoia or later, you can change your background using the built-in functionality of the operating system.
